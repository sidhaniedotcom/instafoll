## Requirements
1. openssl

## How to Install in Termux
```
$ pkg up -y

$ pkg install openssl-tool

$ pkg install curl

$ pkg install git

$ git clone https://github.com/sidhaniedotcom/instafoll

$ cd instafoll

$ chmod +x insfollow.sh

$ termux-wake-lock

$ bash insfollow.sh
```
